# ⚽ Carrom Football Pro

**Carrom Football Pro** is a high-fidelity, physics-based sports game that blends the precision of Carrom with the strategic positioning of Football. Built with high-performance HTML5 Canvas and vanilla JavaScript, it offers a smooth, "board-game" inspired experience directly in your browser.

[![Live Game](https://img.shields.io/badge/Live-Play%20Now-brightgreen?style=for-the-badge)](https://carrom-football-pro.vercel.app/)

---

## 🎮 Game Features

- **Strategic Passing System**: Successfully passing to a teammate triggers an "Extra Turn" and repositions the receiver for a perfect follow-up shot.
- **Advanced AI**: Play against three difficulty levels (Easy, Medium, Hard) that use tactical logic to block goals and execute combos.
- **Multiplayer**: Challenge a friend in Local 2-Player mode on the same device.
- **Mobile-First Design**: Automatically detects orientation (Portrait/Landscape) and scales the board for the best experience on phones, tablets, and desktops.
- **Zero-Latency Physics**: Realistic collision detection, friction, and restitution (bounce) tuned for a professional feel.

## ✨ Visual Aesthetic

- **Board Game Style**: Clean 2D graphics with a striped pitch, pin-marker goal posts, and star-icon pieces.
- **Modern UI**: Pill-shaped badges, glassmorphism overlays, and smooth CSS transitions.
- **Dynamic Orientation**: The board layout repositions itself dynamically when you rotate your device.

## 🛠️ Tech Stack

- **Core**: Vanilla JavaScript (ES6+)
- **Graphics**: HTML5 Canvas API
- **Styling**: CSS3 (with Custom Properties)
- **Deployment**: Vercel (Global CDN)

## 🚀 Deployment

The project is hosted on Vercel and automatically scales to handle thousands of concurrent players.
